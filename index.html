Manual de Referencia Astro

¡Bienvenido a mi manual de reference Astro!

Este manual técnico detalla el desarrollo de un sitio web estático multipágina creado con Astro, el cual sirve como referencia para aplicar conceptos de layouts, componentes, pages y estilos con Tailwind CSS. La guía abarca desde los requisitos previos e instalación en Linux utilizando un proyecto de ejemplo, hasta las extensiones necesarias.
